# Algorithm-Visualizer (Console)

Визуализатор алгоритмов поиска и сортировки, пошагово или автоматически с выбранным размером коллекции.

[![.NET CI](https://github.com/cut9/Algorithm-Visualizer/actions/workflows/dotnet-ci-windows.yml/badge.svg)](https://github.com/cut9/Algorithm-Visualizer/actions)
[![Release](https://img.shields.io/github/v/release/cut9/Algorithm-Visualizer)](https://github.com/cut9/Algorithm-Visualizer/releases)

---

## Информация
- Платформа: кроссплатформенная консоль (.NET)
- Язык: C# .NET 8
- Тип: Консольное приложение (CLI)

---

## Быстрый старт — запустить релиз (рекомендуется для обычных пользователей)

1. Перейдите в раздел **Releases**: https://github.com/cut9/Algorithm-Visualizer/releases  
2. Скачайте архив `win-x64.rar` для вашей платформы.  
3. Распакуйте архив и дважды кликните `Algorithm Visualizer.exe`.

---

## Быстрый старт — запустить из исходников (для разработчиков)

### Команды (PowerShell / CMD)
```powershell
git clone https://github.com/cut9/Algorithm-Visualizer.git
cd Algorithm-Visualizer
dotnet restore
dotnet run --project ./"Algorithm Visualizer"/"Algorithm Visualizer.csproj"
```
